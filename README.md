# scorm_master

A new Flutter project.

## Configuración de Desarrollo

1. Crea un archivo `secrets.json` en la raíz del proyecto con tu clave `GEMINI_API_KEY`.
2. Inicia el proyecto en Linux o macOS ejecutando `./dev.sh`.

Ejemplo de `secrets.json` (colócalo en la raíz del proyecto):

```json
{
  "GEMINI_API_KEY": "TU_CLAVE_AQUI"
}
```

> **Advertencia:** NUNCA subas `secrets.json` al repositorio. Ya está incluido en `.gitignore` por seguridad.
